## MAFFIN

For looking at the structure beneath:
 `@debug`

If you need to print something:
`print( "something" )`

To compile text
`~`

```asm
    @debug

    int ok = 3;
    int op = 2;
    print(ok+op);
    
    ~
```

```asm
    ok 0 3 =  
    op 0 2 =  
    print ok op + (
```
